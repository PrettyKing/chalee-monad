# 🚀 Chalee Monad Demo\n\n一个在 Monad Testnet 上部署和运行的完整智能合约演示项目。这个项目包含了一个 ERC20 代币合约 (ChaleeToken) 以及一个交互式的 Web 界面。\n\n## 📋 项目特性\n\n- ✅ **ERC20 代币合约**: 基于 OpenZeppelin 的安全代币实现\n- ✅ **白名单铸造**: 支持白名单用户的限量铸造功能\n- ✅ **暂停机制**: 合约拥有者可以暂停/恢复合约功能\n- ✅ **Web 界面**: 现代化的 Web3 交互界面\n- ✅ **完整测试**: 全面的智能合约测试套件\n- ✅ **部署脚本**: 自动化的 Monad Testnet 部署\n\n## 🛠️ 技术栈\n\n- **智能合约**: Solidity 0.8.20\n- **开发框架**: Hardhat\n- **前端**: HTML5, CSS3, JavaScript (ES6+)\n- **Web3**: Ethers.js v5\n- **后端**: Node.js + Express\n- **网络**: Monad Testnet\n\n## 📦 安装与设置\n\n### 1. 克隆仓库\n\n```bash\ngit clone https://github.com/PrettyKing/chalee-monad.git\ncd chalee-monad\n```\n\n### 2. 安装依赖\n\n```bash\nnpm install\n```\n\n### 3. 环境配置\n\n复制环境变量模板并填入你的配置：\n\n```bash\ncp .env.example .env\n```\n\n编辑 `.env` 文件：\n\n```bash\n# 你的私钥 (用于部署合约)\nPRIVATE_KEY=your_private_key_here\n\n# Monad Testnet RPC URL (默认即可)\nMONAD_TESTNET_RPC_URL=https://testnet1.monad.xyz\n\n# 区块链浏览器 API Key (可选，用于验证合约)\nMONAD_EXPLORER_API_KEY=your_explorer_api_key_here\n\n# 应用端口\nPORT=3000\n```\n\n⚠️ **安全提示**: 确保你的私钥安全，永远不要提交到 Git 仓库！\n\n### 4. 获取测试代币\n\n访问 [Monad Testnet 水龙头](https://faucet-testnet.monad.xyz) 获取测试用的 MON 代币，用于支付 Gas 费用。\n\n## 🚀 部署与运行\n\n### 1. 编译合约\n\n```bash\nnpm run compile\n```\n\n### 2. 运行测试\n\n```bash\nnpm test\n```\n\n### 3. 部署到 Monad Testnet\n\n```bash\nnpm run deploy:testnet\n```\n\n部署成功后，你会看到类似以下的输出：\n\n```\n✅ ChaleeToken deployed successfully!\n📍 Contract address: 0x1234567890123456789012345678901234567890\n🔗 Transaction hash: 0xabcdef...\n🔍 View on Monad Explorer:\n   https://explorer-testnet.monad.xyz/address/0x1234567890123456789012345678901234567890\n```\n\n### 4. 启动 Web 应用\n\n```bash\nnpm start\n```\n\n应用将在 `http://localhost:3000` 运行。\n\n## 🎯 功能使用指南\n\n### Web 界面功能\n\n1. **连接钱包**: 点击\"Connect Wallet\"连接你的 MetaMask 钱包\n2. **添加网络**: 如果没有 Monad Testnet，点击\"Add Network to Wallet\"\n3. **查看代币信息**: 实时显示合约信息和你的代币余额\n4. **白名单铸造**: 如果你在白名单中，可以铸造代币（最多 1000 CHAL/次）\n5. **转账**: 向其他地址转移代币\n6. **交易历史**: 查看你的交易记录\n\n### 智能合约功能\n\n#### 基础 ERC20 功能\n- ✅ 转账 (`transfer`)\n- ✅ 授权转账 (`transferFrom`)\n- ✅ 查询余额 (`balanceOf`)\n- ✅ 查询授权额度 (`allowance`)\n\n#### 扩展功能\n- ✅ 拥有者铸造 (`mint`) - 仅拥有者\n- ✅ 白名单铸造 (`whitelistMint`) - 白名单用户\n- ✅ 白名单管理 (`updateWhitelist`, `batchUpdateWhitelist`) - 仅拥有者\n- ✅ 暂停/恢复 (`pause`, `unpause`) - 仅拥有者\n- ✅ 查询剩余供应量 (`remainingSupply`)\n\n## 📊 合约详情\n\n### ChaleeToken 参数\n\n- **名称**: Chalee Token\n- **符号**: CHAL\n- **精度**: 18\n- **最大供应量**: 1,000,000 CHAL\n- **初始供应量**: 100,000 CHAL (分配给部署者)\n- **白名单限制**: 每次最多 1000 CHAL，个人总计最多 5000 CHAL\n\n### 网络信息\n\n- **网络名称**: Monad Testnet\n- **Chain ID**: 41454 (0xA1F6)\n- **RPC URL**: https://testnet1.monad.xyz\n- **区块链浏览器**: https://explorer-testnet.monad.xyz\n- **水龙头**: https://faucet-testnet.monad.xyz\n\n## 🧪 测试\n\n项目包含完整的测试套件，覆盖合约的所有功能：\n\n```bash\n# 运行所有测试\nnpm test\n\n# 查看测试覆盖率\nnpx hardhat coverage\n```\n\n测试内容包括：\n- 合约部署\n- ERC20 基础功能\n- 铸造功能\n- 白名单管理\n- 暂停机制\n- 边界条件测试\n\n## 📝 脚本命令\n\n```bash\n# 编译合约\nnpm run compile\n\n# 运行测试\nnpm test\n\n# 部署到测试网\nnpm run deploy:testnet\n\n# 验证合约 (可选)\nnpm run verify\n\n# 启动 Web 应用\nnpm start\n```\n\n## 🔍 验证合约\n\n部署后，你可以选择在区块链浏览器上验证合约源码：\n\n```bash\nnpx hardhat verify --network monadTestnet <CONTRACT_ADDRESS>\n```\n\n## 📂 Wallet\"按钮自动添加。\n\n### Q: 部署失败怎么办？\nA: 检查以下项目：\n- 确保 `.env` 文件中的私钥正确\n- 确保钱包中有足够的 MON 代币支付 Gas\n- 检查网络连接\n\n### Q: 如何获取白名单权限？\nA: 联系合约拥有者，通过 `updateWhitelist` 函数添加你的地址到白名单。\n\n### Q: 可以修改代币参数吗？\nA: 最大供应量、代币名称等参数在部署时确定，无法修改。但拥有者可以管理白名单和暂停状态。\n\n---\n\n**Happy Coding! 🎉**"
